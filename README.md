# Smart Tablature

## User Stories

```
As a Musician
So I can compose music
I would like notes to be displayed as I am playing them
```
```
As a Musician
So I can pick a song
I would like to see a list of songs tablature
```
```
As a Musician
So I can practice
I would like to see a song tablature
```
```
As a Musician
So I can keep track of where I am in the song
I would like the notes to light up as I am playing
```
```
As a Musician
So I can see if I am playing correctly
I want the notes to indicate if I played correctly
```
```
As a Musician
So I don't have to stop playing
I want the tablature to show the part I am about to play
```
